Minha Biblioteca: Uma Webpage Personalizada

Esse projeto é uma página web focada em receitas de brownies que desenvolvi usando HTML e CSS. A página foi criada para compartilhar receitas deliciosas e fáceis, apresentando passo a passo o preparo dos brownies e dicas para um resultado perfeito..

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

A estrutura do site é feita com HTML, que organiza o conteúdo em seções como o título, uma breve introdução, a lista de ingredientes e o modo de preparo. Cada receita é apresentada em um contêiner (.receita) que exibe os ingredientes necessários, uma foto do brownie pronto e instruções detalhadas..

O CSS personaliza a aparência da página, aplicando uma paleta de cores aconchegante e tipografias que deixam o site visualmente agradável. Fontes externas do Google Fonts são usadas para dar um toque especial, e o layout é responsivo para se adaptar a diferentes dispositivos..

Projeto de página inicial de site de assinatura inspirado na Star+

Este projeto é uma página web desenvolvida com HTML e CSS, com o propósito de apresentar uma plataforma de assinatura similar à Star+. A ideia é criar um ambiente visualmente atraente e funcional, onde o usuário possa visualizar o conteúdo disponível e informações de assinatura.

A estrutura do site é planejada com HTML para organizar o conteúdo em seções bem definidas, incluindo um cabeçalho que destaca o nome do serviço, uma divisão principal com uma breve apresentação da plataforma e uma seção de destaque para o catálogo de séries e filmes populares. Cada item de conteúdo é exibido em contêineres próprios, proporcionando uma experiência visual agradável e intuitiva.

O CSS é utilizado para estilizar a página, com ênfase em cores modernas e tipografia que reforçam a identidade inspirada em serviços de streaming. Além disso, o design responsivo permite que a página se adapte a diferentes dispositivos e tamanhos de tela, garantindo uma experiência coesa e envolvente. Esse uso de HTML e CSS oferece uma solução acessível e eficiente para quem busca uma interface de fácil navegação e atrativa.

Flash Card: Ferramenta Interativa de Estudo

Este projeto é um site interativo de perguntas e respostas desenvolvido no Scratch, criado para fornecer uma experiência de aprendizado prática e divertida. O site apresenta cartões com perguntas e respostas, permitindo que os usuários testem seus conhecimentos de forma dinâmica e envolvente.

A estrutura é simples e acessível, com cartões organizados para que cada pergunta possa ser revelada ao clicar, mostrando a resposta correspondente. O design do Scratch utiliza animações e botões interativos, facilitando a navegação e tornando o aprendizado intuitivo e acessível a todas as idades.

Com o Scratch, o projeto aproveita blocos de programação para controlar a interação entre os cartões e os usuários, criando uma plataforma educativa que torna o aprendizado mais atraente e acessível
